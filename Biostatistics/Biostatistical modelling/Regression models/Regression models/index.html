Regression models

Background

Regression models are very powerful mathematical models which can be used in predictive and epidemiological studies (however the approach to certain aspects may differ). In general, all regression models are linear and have a simple form of:

\[Y = mx + b \]

Examples of regression models are as follows:

  1. Simple Linear Regression
  2. Multiple linear regression
  3. Simple Logistic Regression
  4. Multiple Logistic Regression
  5. Poisson Regression
  6. Cox Proportional Hazards model
  7. Exponential and Weibull survival models

Regression models are exciting as there are additional features and considerations to add to these basic models. See Advanced Biostatistical Modelling

How to choose a regression model

Questions to ask:

  1. Is my outcome continuous (i.e. numerical in Real number space?)
    1. Is there a single predictor variable? - use Simple Linear Regression
    2. Are there multiple predictor variables? - use Multiple linear regression
  2. Is my outcome binary (i.e. yes/no, dead/alive)?
    1. Is there a single predictor variable? - use Simple Logistic Regression
    2. Are there multiple predictor variables? use Multiple linear regression
  3. Is the study based on time-to-event data?
    1. Can we make a parametric assumption regarding the outcome?

Assumptions of the regression model

We can't directly use the regression models as they are - we need to assess their assumptions! See Assumptions of regression models

Caveats in drawing conclusions

Take the following into account when drawing conclusions:

  1. Avoid extrapolating beyond the data range - we can only be assured the association holds true within the range you have plotted
  2. An association does not imply causation - it only implies correlation.
