Saving serialised data

Background

Data generated in R can be saved (e.g. dataframes, lists, etc.) in a serialised format - this is a ?lossless means of storage which allows you to store the oriignal data is as accurate a form as possible. Data can also be saved in .csv format, or even .txt, which allows access to the data via other applications (e.g. MS Excel, notepad, etc.).

Code

saveRDS(x, file = "myvector_serialized.rds")

Example

saveRDS(includedPatients, file = "includedPatients.rds")

Loading the data

Data can then be loaded in the following manner:

x <- readRDS("myvector_serialized.rds")
